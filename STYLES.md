## 🔥 Aktuella Mega-trender

### 1. **Neo-Brutalism**
**Beskrivning:** Rå, ofärdig estetik med tjocka svarta borders, klumpiga former, system-fonts, och avsiktligt "fula" färgkombinationer. Motsatsen till polish.
- **Nyckeldrag:** Tjocka svarta outlines (`border-4 border-black`), hårda skuggor (`box-shadow: 4px 4px 0 black`), pixliga ikoner, system-fonts (Arial, Comic Sans), oväntade färgkrockar
- **Palette:** Knallgult + cyan + magenta på vitt, eller neon på svart
- **Typos:** Comic Sans, Arial Black, system-monospace
- **Vibe:** Rå, rebellisk, DIY, anti-corporate, konstnärlig
- **Passar dig?** ⭐⭐⭐⭐⭐ **PERFEKT** för en konstnär! Brutalism = artist alley aesthetic. Det känns handgjort, autentiskt, och sticker ut.
- **Sök:** `neo-brutalism web design`, `brutalist portfolio`, `neo-brutalism CSS`
- **Exempel:** [Figma](https://www.figma.com), [Dropbox Design](https://dropbox.design)

---

### 2. **Kawaii / Soft UI / "Bento Box"**
**Beskrivning:** Rundade former, pastellfärger, mjuka skuggor, lekfulla illustrationer. Som en digital ask med godis.
- **Nyckeldrag:** `rounded-3xl`, pastellgradienter, klistermärkes-estetik, bubbliga knappar, lekfulla mikro-animationer (wiggle, bounce)
- **Palette:** Babyrosa, lavendel, mintgrön, mjuk gul
- **Typos:** Runda sans-serifs (Quicksand, Nunito, Varela Round)
- **Vibe:** Söt, lekfull, välkomnande, vänlig
- **Passar dig?** ⭐⭐⭐⭐⭐ **PERFEKT** för anime/konstnär! Det är det vi nästan byggde, men med mer karaktär.
- **Sök:** `kawaii web design`, `soft UI design`, `bento box UI`, `cute portfolio`
- **Exempel:** [Stripe](https://stripe.com) (bento grids), [Linear](https://linear.app) (soft UI)

---

### 3. **Cyberpunk / Neon-Dark / "Tokyo Night"**
**Beskrivning:** Mörka bakgrunder med lysande neonlyor, skarpa kontraster, futuristiska element. Som Blade Runner fast på webben.
- **Nyckeldrag:** Mörk bakgrund (`#0a0a0f`), neonaccenter (cyan, magenta, gul), glödeffekter, grid-mönster, terminal-text, datamoshing
- **Palette:** Svart/mörkblå + neonrosa + elektrisk cyan
- **Typos:** Monospace + futuristiska sans-serifs (Orbitron, Rajdhani)
- **Vibe:** Futuristisk, edgy, tech-noir, nattliv
- **Passar dig?** ⭐⭐⭐ **Kan funka** om din konst har sci-fi/anime-action tema. Mindre passande för gullig konst.
- **Sök:** `cyberpunk web design`, `neon dark theme`, `tokyo night aesthetic`
- **Exempel:** [Cyberpunk 2077](https://cyberpunk.net), [GitHub Dark](https://github.com)

---

### 4. **Y2K / Frutiger Aero / Webcore**
**Beskrivning:** Nostalgi för tidigt 2000-tal. Glänsande ytor, reflektioner, glas-effekter, blommor, fjärilar, regnbågar. MySpace-estetik möter modern webb.
- **Nyckeldrag:** Glasmorfism (transparens + blur), reflektioner, glittrande ikoner, Comic Sans, sparkles, regnbågsgradienter, hundöron på bilder, "rawr"-estetik
- **Palette:** Ljusblå + silver + lime, eller rosa + lila + turkos
- **Typos:** Georgia, Comic Sans MS, Arial, Times
- **Vibe:** Nostalgisk, ironisk, maximalistisk, DIY, Tumblr 2012
- **Passar dig?** ⭐⭐⭐⭐⭐ **OTROLIGT PASSANDE!** Anime-fandom = Y2K-nostalgi. Sparkles, glitter, hundöron, "rawr xD" energy.
- **Sök:** `Y2K web design`, `frutiger aero`, `webcore aesthetic`, `2000s internet aesthetic`
- **Exempel:** [Neocities](https://neocities.org), [SpaceHey](https://spacehey.com)

---

### 5. **Handgjord / Skissad / "Art Directors Portfolio"**
**Beskrivning:** Ser ut som den är handritad eller skissad. Ojämna linjer, papperstexturer, handskrivna fonts, "ofärdiga" kanter.
- **Nyckeldrag:** Ojämna borders, texturer (papper, akvarell), handskrivna rubriker, collage-layout, mixade media, "skissade" illustrationer
- **Palette:** Papper-vitt + akvarellfläckar, eller mörkt med kritstreck
- **Typos:** Handskrivna fonts (Permanent Marker, Caveat), display-serifs med karaktär
- **Vibe:** Konstnärlig, autentisk, handgjord, personlig
- **Passar dig?** ⭐⭐⭐⭐⭐ **PERFEKT** för en konstnär! Visar att du faktiskt ritar.
- **Sök:** `hand drawn web design`, `sketchy portfolio`, `art director portfolio`, `collage web design`
- **Exempel:** [Tavi Gevinson](https://taviogevinson.com), [Dani Roche](https://danielle.roche.com)

---

### 6. **Anti-Design / Maximalism / "Chaotic Good"**
**Beskrivning:** Bryter ALLA regler. Flera fonts, färger som krockar, oväntade layout, rörliga element, kaos. Men med avsikt.
- **Nyckeldrag:** Flera färgscheman på samma sida, 5+ fonts, diagonala linjer, överlappande element, konstant rörelse, inga marginaler
- **Palette:** ALLA FÄRGER. Samtidigt.
- **Typos:** ALLA FONTER. Samtidigt.
- **Vibe:** Energisk, uttrycksfull, ung, rebellisk, konstnärlig frihet
- **Passar dig?** ⭐⭐⭐⭐ **BRA** om du vill visa att du inte följer regler. Risk: svårt att navigera.
- **Sök:** `anti-design web`, `maximalist web design`, `chaotic good aesthetic`, `break the grid web`
- **Exempel:** [OKFocus](https://okfoc.us), [Experiments by Google](https://experiments.withgoogle.com)

### 7. **Glassmorphism / Aurora UI**
**Beskrivning:** Genomskinliga ytor med blur, mjuka gradienter som dansar i bakgrunden. Känns som att titta genom frostat glas.
- **Nyckeldrag:** `backdrop-blur-xl`, `bg-white/10`, mjuka gradient-bollar, subtila borders, djup
- **Palette:** Mörk bakgrund + mjuka pastell-gradienter (rosa→lila→blå)
- **Typos:** Clean sans-serifs (SF Pro, Inter, Geist)
- **Vibe:** Elegant, modern, premium, "Apple-liknande"
- **Passar dig?** ⭐⭐⭐ **Okej** men kan kännas för "corporate" för en konstnär. Bra som accent.
- **Sök:** `glassmorphism CSS`, `aurora UI`, `frosted glass web design`
- **Exempel:** [Apple](https://apple.com), [Microsoft Fluent](https://fluent2.microsoft.design)

---

### 8. **Pixel Art / Retro Gaming / 8-bit**
**Beskrivning:** Som ett SNES-spel. Pixel-perfect, begränsad färgpalett, retro-ikoner, chiptune-känsla.
- **Nyckeldrag:** Pixliga kanter, 16-färgers palett, pixel-fonts, sprite-ikoner, scanlines, CRT-effekt
- **Palette:** NES-palett eller begränsade färgscheman
- **Typos:** Press Start 2P, VT323, pixel-fonts
- **Vibe:** Nostalgisk, gaming, indie, retro
- **Passar dig?** ⭐⭐⭐⭐ **BRA** om din konst inkluderar pixel art, gaming-tema, eller chibi-stil.
- **Sök:** `pixel art web design`, `8-bit portfolio`, `retro gaming aesthetic`
- **Exempel:** [Poolside FM](https://poolside.fm), [Carrd](https://carrd.co) (vissa templates)

---

### 9. **Vaporwave / Seapunk / "Aesthetic"**
**Beskrivning:** 90-talsnostalgi med grekiska statyer, rosa/svarta gradienter, palmer, solnedgångar, kassettband, Windows 95.
- **Nyckeldrag:** Grekiska statyer, rosa→blå gradienter, checkerboard-mönster, Wingdings, Helvetica, kassettband, 3D-text med krom
- **Palette:** Rosa + ljusblå + violett + svart + turkos
- **Typos:** Helvetica, VCR OSD Mono, Times
- **Vibe:** Ironisk, nostalgisk, surrealistisk, "aesthetic" meme-kultur
- **Passar dig?** ⭐⭐⭐ **ROLIGT** men kanske för ironiskt? Bra för speciella projekt eller områden.
- **Sök:** `vaporwave aesthetic`, `seapunk design`, `sad boys aesthetic web`
- **Exempel:** [Vaporwave.pl](https://vaporwave.pl), [Society6](https://society6.com) (vissa kategorier)

### 10. **Monospace / Terminal / "Dev Aesthetic"**
**Beskrivning:** Ser ut som en kod-editor eller terminal. Grön text på svart, monospace överallt, blockiga element, utilitarian.
- **Nyckeldrag:** Monospace fonts, grön text (`#00ff00`), block borders, ingen fluff, utilitarian layout, "raw" text
- **Palette:** Svart + grön/amber/cyan + grå
- **Typos:** Mono: JetBrains Mono, Fira Code, Space Mono
- **Vibe:** Tech, programmerare, h4x0r, utilitarian
- **Passar dig?** ⭐ **Inte så bra** för en konstnär sida. Känns för steril.
- **Sök:** `terminal aesthetic`, `monospace portfolio`, `hacker aesthetic`
- **Exempel:** [Gucci](https://gucci.com) (vissa kampanjer), [Linus Lee](https://thesephist.com)

### 11. **Bauhaus / Swiss Design / "Brutalist Grid"**
**Beskrivning:** Ren, geometrisk, grid-baserad. Primärfärger (röd, gul, blå), Helvetica, mycket whitespace.
- **Nyckeldrag:** Strikta grids, primärfärger, geometriska former, asymmetriska layouter, mycket whitespace
- **Palette:** Röd + gul + blå + svart + vitt
- **Typos:** Helvetica, Akzidenz Grotesk, Univers
- **Vibe:** Modernistisk, akademisk, ren, intellektuell
- **Passar dig?** ⭐⭐ **Kanske inte.** För ren för anime. Men element kan lånas.
- **Sök:** `bauhaus web design`, `swiss design grid`, `international typographic style`
- **Exempel:** [Pentagram](https://pentagram.com), [Swiss Style Now](https://swissstylenow.com)

### 12. **Organic / Nature / "Blooming"**
**Beskrivning:** Mjuka former som påminner om natur. Blommor, löv, organiska kurvor, jordtoner, handgjort papper.
- **Nyckeldrag:** Organiska former (blobbs), jordtoner, blom-illustrationer, handgjorda texturer, växtmotiv
- **Palette:** Terracotta, olivgrön, sand, djup skogsgrön, cremevit
- **Typos:** Serif display fonts (DM Serif, Playfair Display), handskrivna
- **Vibe:** Naturlig, lugn, hållbar, bohemisk
- **Passar dig?** ⭐⭐⭐ **Kan funka** om din konst inkluderar natur, blommor, fantasy-skogar.
- **Sök:** `organic web design`, `nature aesthetic portfolio`, `boho web design`
- **Exempel:** [Patagonia](https://patagonia.com), [Moon Juice](https://moonjuice.com)

## 🎯 Rekommendation för dig: "Neo-Brutalism + Y2K + Handgjort"

Min personliga rekommendation är en **hybrid** som kombinerar det bästa av dessa tre:

1. **Neo-Brutalism** för råa, artistiska kanter och DIY-känsla
2. **Y2K** för sparkles, regnbågar, glitter, och anime-nostalgi
3. **Handgjort** för att visa att du faktiskt ritar

### Så här skulle det se ut:
- **Bakgrund:** Antingen papperstextur VIT, eller mörk (#1a0a2e) med neonaccenter
- **Borders:** Tjocka, handritade känsla (inte perfekta)
- **Färger:** Knalliga regnbågsfärger mot vitt, eller neon på svart
- **Font:** En quirky display-font (t.ex. "Fredoka One", "Chewy") + en handskriven
- **Element:** Sparkles, glitter, stickers som flyger omkring, hundöron på bilder, "klistermärkes"-känsla
- **Layout:** Lite "kaotisk" men funktionell — som en scrapbook eller en artist alley-bord

### Exempel att studera:
1. **Goblin Queen** (om du kan hitta) — typisk artist alley aesthetic
2. **Society6 artist shops** — se hur konstnärer presenterar sina verk
3. **Tumblr teman från 2012** — maximalistisk, personlig, glitter
4. **FurAffinity / DeviantArt profiles** — råa, personliga, konstnärliga
5. **Japanese web design** — ofta quirky, färgglada, annorlunda än västerländsk minimalism
